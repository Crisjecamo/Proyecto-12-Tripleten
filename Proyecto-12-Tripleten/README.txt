Proyecto 12 - Tripleten: Automatización

Descripción del Proyecto:
Se proporciona un dataset llamado trending_by_time.csv, el cual contiene información detallada sobre vídeos publicitarios en la agencia de publicidad Sterling & Draper. 
Esta información es utilizada para analizar tendencias de vídeos en YouTube y ayudar a determinar qué contenido es relevante para las campañas de marketing.

Objetivo:
El objetivo es crear un dashboard interactivo en Tableau que presente los contenidos y datos acordados con el equipo de marketing. 
Este dashboard servirá para visualizar las tendencias de manera clara y accesible para la toma de decisiones.


Pasos para ejecutar los archivos

Acceso al Dashboard en Tableau Public:

Haz clic en el Link https://acortar.link/BBdNeG Esto abrirá la página web del dashboard en el servidor de Tableau Public.
En caso de que la página web no se abra dando clic copia el link y pegalo directamente en tu navegador o en el principio de este repositorio abajo esta el link completo que los llevara al dashboard.

Presentación PDF:

Abre el archivo videos-en-tendencia-proyecto-12-tripleten.pdf. Este documento contiene una presentación con los resultados más importantes generados por el dashboard, los cuales son de interés para el equipo de marketing.
La presentación destaca los análisis más relevantes y proporciona un resumen visual de las tendencias clave para los vídeos publicitarios, facilitando la interpretación y toma de decisiones del equipo.
